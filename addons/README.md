# Addons

This folder contains optional addons for Productive K3S.

Each addon should live in its own folder and should include, at minimum, a short README explaining:

- what the addon does
- whether it is experimental, community, or supported
- how it is installed
- what dependencies it has
- how it relates to `productive-k3s-core` or `productive-k3s-infra`

Addons are not part of the core runtime.
