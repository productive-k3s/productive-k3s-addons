# Relationship with Core and Addons Pro

`productive-k3s-core` is the engine. It discovers add-on metadata, resolves stack membership, and executes normalized hooks.

`productive-k3s-addons` contains:

- `addons/`: individually deployable capabilities
- `stacks/`: declarative collections of add-ons

`productive-k3s-addons-pro` follows the same repository contract for private content.
