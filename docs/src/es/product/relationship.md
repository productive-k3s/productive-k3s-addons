# Relación con Core y Addons Pro

`productive-k3s-core` es el engine. Descubre metadata de add-ons, resuelve la pertenencia a stacks y ejecuta hooks normalizados.

`productive-k3s-addons` contiene:

- `addons/`: capacidades desplegables individualmente
- `stacks/`: colecciones declarativas de add-ons

`productive-k3s-addons-pro` sigue el mismo contrato de repositorio para contenido privado.
